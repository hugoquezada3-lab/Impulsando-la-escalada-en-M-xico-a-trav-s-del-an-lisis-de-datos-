# Impulsando-la-escalada-en-M-xico-a-trav-s-del-an-lisis-de-datos-
"Impulsando la escalada en México a través del análisis de datos" collects and organizes information about climbing areas, types of climbing, and geographical locations. The project uses Tableau, SQL, Excel, and Jira to generate visualizations and insights that promote the development of climbing in the country.
